CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using alink element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in the external CSS files. However, in this tutorial we will use inline nad internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

when comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements, ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL tolink directly to an element and it's used by JavaScript.

In CSS, selectorsareused to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class ad ID. Both are used to target elements to which a style should be applied.
